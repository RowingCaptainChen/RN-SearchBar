# ReactNative-SearchBar
---
The highly customized search bar for React Native
---

---
###Usage
`
<SearchBar searchAction={()=>{
                    Alert.alert('searchAction')
                }}
                   onPressCancel={()=>{
                    Alert.alert('onPressCancel')
                }}
                    borderRadius={6}
                    text={'Search'}
                    rightText={'Cancel'}
                    containerColor={'blue'}
/>
`
